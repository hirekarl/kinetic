# Kinetic Design System

**Aesthetic:** Developer Tool Minimalism (Vercel, Linear, Stripe inspired)
**Framework:** Tailwind CSS

## Core Principles
1. **Typography & Hierarchy:** Rely on font-weight and subtle color contrast (`text-zinc-400` vs `text-white`) rather than wildly varying sizes.
2. **Layout:** Split-panel, full-height Flexbox/Grid. 1px subtle borders (`border-zinc-800`), no heavy drop shadows.
3. **Status Indicators:** Small, crisp, semantic colored dots (emerald-500, amber-500, rose-500) over large colored blocks.
4. **Surfaces:** Deep muted grays (`bg-zinc-950`, `bg-zinc-900`) for surfaces to let the status colors pop.

## Key Colors
* **Background:** `zinc-950`
* **Surface/Card:** `zinc-900/50`
* **Border:** `zinc-800`
* **Text (Primary):** `zinc-100`
* **Text (Muted):** `zinc-400`
* **Status Green:** `emerald-500`
* **Status Yellow:** `amber-500`
* **Status Red:** `rose-500`
