Sleep, domestic logistics, and close relationships accumulate hidden debt with no alerting and no triage list. The only feedback loop is burnout, which fires weeks after the damage is done.

Key Technical Decisions
Structured outputs via Instructor. Forces Gemini 2.5 Flash to return typed, validated Pydantic data — clean payload or loud error. No silent JSON drift.
Parallel agent dispatch. All three agents fire simultaneously and merge at the end. Sequential execution would add two extra round-trips per check-in.
Dual-mode persistence. A DatabaseClient Protocol abstracts SQLite (local dev, zero config) and asyncpg PostgreSQL (Render production). Zero application code changes at deploy time.
SSE over WebSocket. Token streaming via Server-Sent Events using fetch + ReadableStream — EventSource cannot send POST bodies.
